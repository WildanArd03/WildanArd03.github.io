Membuat laporan PDF dengan DOMPDF

Karena saya akan membuat PDF dengan DOMPDF Laravel, maka langkah pertama yang saya akan lakukan adalah membuat route baru untuk menampilkan data kelengkapan dokumen dan lampiran dari database dan menampilkan sebagai PDF.

Berikut contohnya :

Route yang pertama "kelengkapan-dokumen" akan mengakses method index() dalam controller "kelengkapandokumenController.php". Route kedua "kelengkapandokumen-print" kita perintah untuk mengakses method print() dalam controller "kelengkapandokumenController.php". Langkah selanjutnya yaitu membuat file controller.

Sesudah membuat file controller, buka file controller tersebut dan buat kedua method yang sudah disebutkan di atas tadi.

app/Http/Controller/kelengkapandokumenController.php

Pada method yang satunya lagi, yaitu method print() kita ambil semua data kelengkapan dokumen dan kita simpan dalam variabel $kelengkapan_dokumen begitu juga dengan lampiran.

Kemudian buat variabel $pdf untuk PDF::loadview() kita menggunakan dompdf untuk membuka view nya, saya mengisi nama view untuk halaman cetak pdf ini degan view detail.blade.php.
Sambil kita panggil data kelengkapan dokumen dan lampiran ke view ini.

Kemudian bagian terakhir kita retun untuk mencetak halaman menjadi file PDF.
Dan jangan lupa membuat view untuk data data dan tampilannya

Untuk membuat view anda bisa membuatnya di
resources/views/"nama file anda".blade.php
