Membuat form Edit Laravel

Edit merupakan di mana kita ingin mengubah suatu data dengan cara update data tersebut, inti keseluruhan sama saja seperti kita mengubah data pada database pada php native
Hanya saja untuk membuat edit data pada dari database dengan laravel kita harus mengikuti struktur dan perintah perintah yang ada di laravel.

Update Data Dengan Laravel

Sebelumnya saya sudah membuat view untuk menampilkan data kelengkapan dokumen.
Selanjutnya kita buat button edit atau link seperti berikut.

Pada tombol edit tersebut kita memerintahkan untuk mengalihkan halaman ke route "kelengkapandokumen/detail" sambil mengirimkan data id, data yang ingin di edit.
Maka sekarang kita buat route baru untuk edit agar data id yang dikirimkan sekalian kita kirim juga ke controller untuk kita ambil data kelengkapan dokumen yang ber id tersebut.

"nama-project"/routes/web.php

Lalu kita buat method dalam controller kelengkapandokumenController.php

"nama-project"/app/Http/Controllers/kelengkapandokumenController.php

Selanjutnya kita buat view dengan nama detail.blade.php karena kita akan menampilkan data kelengkapan dokumen yang ingin di edit.

"nama-project"/resource/views/detail.php

Dan jangan lupa kita membuat route update agar data yg sudah kita ubah bisa kita simpan
oleh karena itu kita buat route kelengkapandokumen/update di

"nama-project"/routes/web.php

Para route update ini kita tidak menggunakan method get, tapi menggunakan method post
pada route ini kita perintahkan untuk menjalankan method update dalam controller kelengkapandokumenController.php

"nama-project"/app/Http/Controllers/kelengkapandokumenController.php


Lalu kita coba dengan menjalankan xampp dan masuk url localhost:8000/"nama-project"
